BS Grid Basic

Bootstrap's grid system allows up to 12 columns across the page.

The Bootstrap grid system has four classes:

  • xs (for phones)
  • sm (for tablets)
  • md (for desktops)
  • lg (for larger desktops)

Example:

This is grid class .col-sm-4
This is grid class .col-sm-8

BS Typography

  • Bootstrap will style the HTML <code> element as: text inside code
  • Bootstrap will style the HTML <mark> element as: text inside mark
  • Bootstrap will style the HTML <lbd> element as: text inside kbd
  • Bootstrap will style the HTML <samp> element as: This is an example of a computer program ...
  • Bootstrap will style the HTML <var> element as: ax+b=0
  • Bootstrap will style the HTML <pre> element as: 
    This is    a plain text as it       comes.
  • Bootstrap will style the HTML <blockquote> element as:

    For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

    From WWF's website

  • Bootstrap provides the contextual colors classes to offer "meaning through colors":

    <p class="text-muted"> as muted text.

    <p class="text-primary"> as important text.

    <p class="text-success"> as text indicating success.

    <p class="text-info"> as text representing some information.

    <p class="text-warning"> as text representing a warning.

    <p class="text-danger"> as text representing danger.

  • Bootstrap also provides the contextual background-colors classes with the same meaning:

    <p class="text-primary"> as important text.

    <p class="text-success"> as text indicating success.

    <p class="text-info"> as text representing some information.

    <p class="text-warning"> as text representing a warning.

    <p class="text-danger"> as text representing danger.
